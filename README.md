# LimeDFB

This project provides a set of reusable digital function blocks (IP cores), which were created to support the development of [FPGA gateware for the LimeSDR family of boards](https://myriadrf.org/projects/limesdr-gw/). However, they may also prove useful in other applications.

Detailed documentation is available in [https://myriadrf.org/projects/limedfb/](https://myriadrf.org/projects/limedfb/)
